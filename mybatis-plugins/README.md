# mybatis-plugins
## 功能实现
1. 实现自动填充
2. 实现枚举处理
3. 时间类型通用返回和接受
4. 软删除，使用mybatis-plus的方法不物理删除，不修改删除的对象
5. 解决前端可能存在的精度丢失问题
6. 动态显示enum可用值
7. 将enum转换为Json返回给前端
8. 集成knife4j 作为 接口文档
9. 集成knife4j增强模式，支持功能请查阅![knife4j](https://xiaoym.gitee.io/knife4j/documentation/enhance.html) 
10. 支持许可值设置为枚举 告知前端枚举各项而不用手写文档
